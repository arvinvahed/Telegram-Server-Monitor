🇮🇷 Server Monitor Bot: مانیتورینگ هوشمند سرور از طریق تلگرام

⚡️ چرا این مانیتورینگ برای سرور شما مناسب است؟

آیا از ابزارهای مانیتورینگ سنگین و پیچیده خسته شده‌اید؟ Server Monitor Bot یک ابزار فوق‌العاده سبک، سریع و مستقل است که با زبان Go کامپایل شده و بدون نیاز به هیچ وابستگی خارجی (Dependency) یا تنظیمات پیچیده، مستقیماً از طریق تلگرام وضعیت حیاتی سرور شما را در لحظه گزارش می‌دهد.

مزایای کلیدی:

مزیت

توضیح

سبک و مستقل (Binary)

حجم بسیار کم، عدم نیاز به Python, PHP یا Node.js. این برنامه یک فایل اجرایی واحد است.

اجرای پایدار (Go)

نوشته شده با زبان Go برای پایداری و عملکرد بالا. به‌صورت بهینه منابع سرور را مصرف می‌کند.

فقط خروجی (Code Protection)

سورس کد پروژه (Go) عمومی نیست. شما تنها فایل اجرایی را دریافت می‌کنید که امنیت و مالکیت معنوی کد را حفظ می‌کند.

گزارش‌دهی هوشمند

ارسال هشدار فوری فقط زمانی که مصرف CPU، RAM یا دیسک از آستانه‌های خطرناک عبور کند.

🔍 ویژگی‌های مانیتورینگ

این برنامه به‌طور خودکار هر ۳۰ ثانیه موارد زیر را بررسی و گزارش می‌کند:

سلامت منابع: درصد مصرف CPU، RAM و فضای دیسک.

زمان فعال بودن (Uptime): مدت زمان روشن بودن سرور.

میانگین بار (Load Avg): گزارش لود سرور در بازه‌های ۱، ۵ و ۱۵ دقیقه.

مانیتورینگ شبکه: قابلیت دسترسی به اینترنت (Ping به DNS).

چک کردن پورت‌ها: وضعیت باز بودن پورت‌های حیاتی (مانند HTTP، HTTPS، SSH و...) در سطح لوکال سرور.

🛠️ راهنمای نصب کامل روی سرور Ubuntu

نصب این ابزار بسیار ساده و سریع است و فقط با چند دستور انجام می‌شود.

پیش‌نیاز: شما باید توکن یک ربات تلگرام و Chat ID شخصی خود را داشته باشید.

مرحله ۱: دانلود فایل اجرایی (Binary)

وارد ترمینال سرور Ubuntu خود شوید و دستورات زیر را به ترتیب اجرا کنید:

# ۱. ایجاد پوشه مخصوص برنامه و ورود به آن
mkdir telegram-monitor
cd telegram-monitor

# ۲. دانلود فایل اجرایی (Binary) با استفاده از لینک مستقیم Releases
# **⚠️ لطفا لینک دانلود مستقیم فایل server-monitor از Releases خود را جایگزین کنید**
wget <لینک دانلود مستقیم فایل server-monitor از Releases گیت‌هاب شما> -O server-monitor

# ۳. دادن مجوز اجرا (ضروری برای لینوکس)
chmod +x server-monitor


مرحله ۲: تنظیمات تلگرام (از طریق متغیرهای محیطی)

این فایل اجرایی اطلاعات اتصال به تلگرام را از متغیرهای محیطی سیستم عامل می‌خواند. لطفا مقادیر خود را جایگزین کنید:

# ۱. توکن ربات تلگرام خود را جایگزین کنید
export TELEGRAM_BOT_TOKEN="Your_Bot_Token_Here" 

# ۲. آی‌دی چت خود را جایگزین کنید (برای دریافت پیام‌ها)
export TELEGRAM_CHAT_ID="Your_Chat_ID_Here" 


مرحله ۳: اجرای دائمی برنامه

دستور زیر برنامه را در پس‌زمینه (Background) اجرا می‌کند تا حتی پس از بستن ترمینال نیز به کار خود ادامه دهد. خروجی برنامه در فایل servmon.log ذخیره می‌شود.

# اجرای برنامه در پس‌زمینه با nohup
nohup ./server-monitor > servmon.log 2>&1 & 

echo "✅ مانیتورینگ سرور با موفقیت آغاز شد و اکنون فعال است."


🛑 راهنمای توقف و حذف کامل

الف) توقف برنامه

برای متوقف کردن برنامه و جلوگیری از ارسال پیام‌های بیشتر:

pkill server-monitor 

echo "🛑 برنامه Server Monitor با موفقیت متوقف شد."


ب) حذف کامل

برای حذف کامل فایل‌ها و پوشه‌های برنامه از سرور:

# ۱. توقف برنامه (اگر در حال اجراست)
pkill server-monitor 

# ۲. رفتن به پوشه قبلی
cd ..

# ۳. حذف کامل پوشه برنامه و محتویات آن
rm -rf telegram-monitor 

# ۴. حذف فایل لاگ (اگر در پوشه اصلی اجرا شده باشد)
rm -f servmon.log

echo "🗑️ Server Monitor به صورت کامل از سرور حذف شد."
