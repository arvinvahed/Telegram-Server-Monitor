🚀 Server Monitor Bot: قلب تپنده سرور شما در دستان تلگرام!

مفتخریم که Server Monitor Bot را معرفی کنیم؛ ابزاری که مانیتورینگ سرور را از یک کار پیچیده، به یک تجربه لذت‌بخش و لحظه‌ای تبدیل می‌کند.

✨ چرا باید از این ربات استفاده کنید؟

ما می‌دانیم زمان شما چقدر ارزشمند است. این ربات با زبان قدرتمند Go نوشته شده و به صورت یک فایل اجرایی فوق‌العاده سبک و مستقل عرضه می‌شود.

امنیت کد (Binary-Only): 🛡️ سورس کد اصلی (Go) در اختیار عموم قرار نمی‌گیرد. شما تنها فایل کامپایل شده را دانلود می‌کنید و امنیت فکری پروژه ما تضمین شده است.

سریع و سبک: 🏎️ بدون نیاز به نصب پکیج‌های اضافی (مثل Python، PHP و...) یا درگیر شدن با تنظیمات پیچیده. فقط دانلود کن و اجرا کن!

هشدار هوشمند: 🔔 ربات فقط زمانی که مصرف منابع از حد مجاز عبور کند (مانند پر شدن CPU یا RAM)، بلافاصله شما را با یک پیام هشدار مطلع می‌کند.

🛠️ راهنمای نصب و راه‌اندازی سریع (ویژه Ubuntu)

پیش‌نیازها:

یک سرور با سیستم عامل Ubuntu (یا Debian).

توکن ربات تلگرام و Chat ID شخصی خود.

مرحله ۱: دانلود فایل اجرایی 📥

دستورات زیر را در ترمینال سرور خود اجرا کنید تا برنامه دانلود و آماده اجرا شود:

# ۱. ساخت پوشه برای نظم بیشتر و ورود به آن
mkdir telegram-monitor
cd telegram-monitor

# ۲. دانلود فایل اجرایی نهایی (لینک مستقیم از Releases)
# این لینک دانلود اختصاصی و معتبر شماست
wget [https://github.com/arvinvahed/Telegram-Server-Monitor/releases/download/v1.0.0/server-monitor](https://github.com/arvinvahed/Telegram-Server-Monitor/releases/download/v1.0.0/server-monitor) -O server-monitor

# ۳. اعطای مجوز اجرا به فایل
chmod +x server-monitor


مرحله ۲: اتصال به تلگرام 🔑

ربات برای اتصال، توکن و چت آی‌دی را از متغیرهای محیطی سیستم عامل می‌خواند. مقادیر زیر را با مشخصات خود جایگزین کنید:

# ۱. توکن ربات تلگرام خود را جایگزین کنید
export TELEGRAM_BOT_TOKEN="Your_Bot_Token_Here" 

# ۲. آی‌دی چت خود را جایگزین کنید (باید آی‌دی عددی باشد)
export TELEGRAM_CHAT_ID="Your_Chat_ID_Here" 


مرحله ۳: اجرای نهایی و دائمی 🎉

برنامه را در پس‌زمینه اجرا کنید. با دستور nohup، حتی پس از قطع شدن اتصال شما به سرور، ربات به کار خود ادامه می‌دهد:

# اجرای برنامه در پس‌زمینه با nohup و ذخیره لاگ در servmon.log
nohup ./server-monitor > servmon.log 2>&1 & 

echo "✅ مانیتورینگ سرور شما با موفقیت فعال شد! پیام‌های وضعیت را در تلگرام چک کنید."


🛑 توقف و پاکسازی کامل

توقف موقت ربات:

اگر نیاز به توقف لحظه‌ای ربات دارید:

pkill server-monitor 
echo "🛑 برنامه Server Monitor متوقف شد."


حذف کامل و نهایی:

برای حذف کلی تمام فایل‌ها و پوشه‌ها:

# ۱. توقف نهایی برنامه (برای اطمینان)
pkill server-monitor 

# ۲. رفتن به پوشه قبلی
cd ..

# ۳. حذف کامل پوشه برنامه و محتویات آن
rm -rf telegram-monitor 
rm -f servmon.log # حذف فایل لاگ در صورتی که در پوشه اصلی ایجاد شده باشد

echo "🗑️ Server Monitor به صورت کامل از سرور حذف شد. سرور شما پاک است!"
